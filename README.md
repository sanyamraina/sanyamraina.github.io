# Sanyam Raina — Portfolio

Bold editorial personal portfolio built with Next.js and Tailwind CSS.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Getting Started
Install dependencies:
```bash
npm install
```

Run the dev server:
```bash
npm run dev
```

Open http://localhost:3000

## Scripts
- `npm run dev` — start local dev server
- `npm run build` — production build
- `npm run start` — run production server
- `npm run lint` — lint

## Customize Content
Update the placeholder content in:
- `app/page.tsx`

Global styles and theme tokens:
- `app/globals.css`

## Deploy
Deploy on Vercel:
1) Push this repo to GitHub
2) Import the repo on Vercel
3) Use the default Next.js settings

## Notes
If you see a Tailwind native binary error, re-run:
```bash
npm install
```
